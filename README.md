# 🚀 Real-Time Collaborative Code Editor

A live collaborative coding platform where multiple users can edit code and chat in real-time.

## ✨ Features
- Real-time code synchronization
- Room-based collaboration
- Live chat system
- Scalable WebSocket architecture

## 🛠 Tech Stack
- React.js
- Node.js + Express
- Socket.io
- CodeMirror (optional)

## ⚡ Installation

### Backend
cd server
npm install
node server.js

### Frontend
cd client
npm install
npm start
